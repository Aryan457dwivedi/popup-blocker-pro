# Popup Blocker Pro

A Chrome ad blocker built using Manifest V3.

## Features
- Network-level ad blocking
- Live blocked request counter
- Enable / disable toggle
- Badge counter on extension icon
- Lightweight and fast
- Uses Manifest V3 APIs

## Installation

1. Clone the repository

```bash
git clone https://github.com/Aryan457dwivedi/popup-blocker-pro.git
```



2. RUN LOCALLY ON CHROME

  Step 1: Open Chrome and go to: chrome://extensions/

  Step 2: Turn ON Developer mode (top-right corner).

  Step 3: Click Load unpacked.

  Step 4: Select this folder: popup-blocker-pro

  Step 5: Your extension will now appear in the list 

  Step 6: Click the Extension icon, Pin Popup Blocker Pro, and click the extension icon to open the popup.


## TEST THE EXTENSION

  Verify Blocking: Open any website with lots of ads/popups.
  
  Check Stats: Verify that popups are blocked, the counter increases, and the badge number updates.


## Toggle Check

  Turn OFF → ads appear.
  
  Turn ON → ads blocked again.


## AFTER CODE CHANGES
  
  If you make changes to the code, you must reload the extension to see them:
 
  Go to: chrome://extensions/
  
  Click Reload on the Popup Blocker Pro card.
